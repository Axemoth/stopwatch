# Kronos Timer

A sleek futuristic stopwatch and countdown timer built with React + Vite.

## Features

- Stopwatch with lap tracking
- Countdown timer
- Preset timer durations
- Custom timer input
- Animated circular progress dial
- Responsive dark UI
- Smooth animations

## Tech Stack

- React
- Vite
- CSS

## Setup

Create the project:

```bash
npm create vite@latest kronos-timer
```

Choose:

- Framework: React
- Variant: JavaScript

Install dependencies:

```bash
cd kronos-timer
npm install
```

Run the development server:

```bash
npm run dev
```

## Usage

Replace the contents of:

```bash
src/App.jsx
```

with the provided Kronos Timer code.

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Deploy on Vercel

Upload the entire project folder directly to :contentReference[oaicite:0]{index=0}.

Vercel will automatically detect:

- Framework: Vite
- Build Command: `npm run build`
- Output Directory: `dist`

No root folder changes are needed unless using a monorepo setup.
